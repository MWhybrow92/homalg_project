9 Cohomology of coherent sheaves

9.1 Greg Smith Cone

9.1-1 GSCone
‣ GSCone( variety )( attribute )

Returns: a list

Given a smooth and complete toric variety, this method returns a list of hyperplane constraints that define the Greg Smith cone.

9.2 Cohomology of coherent sheaves: truncations of the Cox ring

9.2-1 Exponents
‣ Exponents( variety, degree )( operation )

Returns: a list of lists of integers

Given a smooth and complete toric variety and a list of integers (= degree) corresponding to an element of the class group of the variety, this method return a list of integer valued lists. These lists correspond to the exponents of the monomials of degree in the Cox ring of this toric variety.

9.2-2 MonomsOfCoxRingOfDegreeByNormaliz
‣ MonomsOfCoxRingOfDegreeByNormaliz( variety, degree )( operation )

Returns: a list

Given a smooth and complete toric variety and a list of integers (= degree) corresponding to an element of the class group of the variety, this method returns the list of all monomials in the Cox ring of the given degree. This method uses NormalizInterface.

9.2-3 DegreeXParts
‣ DegreeXParts( variety )( operation )

Returns: a record

Given a toric variety, this method returns a record with the monomials of all degree layers of the Coxring, that have been computed for this toric variety thus far.

9.2-4 DegreeXPart
‣ DegreeXPart( variety, degree )( operation )

Returns: a list

Given a smooth and complete toric variety and a list of integers (= degree) corresponding to an element of the class group of the variety, this method returns the list of all monomials in the Cox ring of the given degree. This method uses NormalizInterface.

9.2-5 DegreeXPartVectorsAsMatrices
‣ DegreeXPartVectorsAsMatrices( variety, degree, i, l )( operation )

Returns: a list

Just as DegreeXPartVects, but the sublists of length l are turned into matrices valued in the Coxring of the toric variety.

9.2-6 DegreeXPartVectorsAsLists
‣ DegreeXPartVectorsAsLists( variety, degree, i, l )( operation )

Returns: a list of lists

Given a smooth and complete toric variety, a list of integers (= degree) corresponding to an element of the class group of the variety, and two non-negative integers i and l, this method returns a list of lists. The sublists are of length l and have at position i the monoms of the Coxring of degree 'degree'.

9.2-7 DegreeXPartOfFreeModule
‣ DegreeXPartOfFreeModule( variety, module, degree )( operation )

Returns: a list of lists

Given a smooth and complete toric variety with Coxring S, a graded free S-module 'module' and list of integers (=degree) corresponding to an element of the class group of the toric variety, this method returns a list of generators of the degree layer of the 'module'.

9.2-8 DegreeXPartOfFreeModuleAsVectorSpace
‣ DegreeXPartOfFreeModuleAsVectorSpace( variety, module, degree )( operation )

Returns: a vector space

Just as DegreeXPartOfFreeModule, but the result is a vector space over the coefficient ring of the Coxring of the variety. Its dimension is the length of the list of generators that the method DegreeXPartOfFreeModule would compute.

9.2-9 DegreeXPartOfFreeModuleAsMatrix
‣ DegreeXPartOfFreeModuleAsMatrix( variety, module, degree )( operation )

Returns: a list of matrices

Just as DegreeXPartOfFreeModule, but the generators are turned into matrices over the Coxring of the variety. Therefore this method returns a list of matrices over the Coxring rather than a list of lists.

9.2-10 DegreeXPartOfFPModule
‣ DegreeXPartOfFPModule( variety, module, degree )( operation )

Returns: a vector space

Given a smooth and complete toric variety with Coxring S, a f.p. graded S-module 'module' and list of integers (=degree) corresponding to an element of the class group of the toric variety, this method computes the degree 'degree' layer of (a) presentation morphisms of 'module' and returns the cokernel object of this homomorphism of vector spaces.

9.3 Cohomology of coherent sheaves: the B-transform

9.3-1 H0FromBTransformInInterval
‣ H0FromBTransformInInterval( variety, M, a, b )( operation )

Returns: a list of vector spaces

Given a smooth and complete toric variety with Coxring S, a f.p. graded S-module M and two non-negative integers a and b, this method computes the degree zero layer of Hom ( B ( x ) , M ) for x ∈ [ a,b ]. In this expression B(x) is the x-th Frobenius power of the irrelevant ideal of the toric variety.

9.4 Cohomology of coherent sheaves: the theorem of Greg Smith

9.4-1 H0ByGS
‣ H0ByGS( variety, M )( operation )

Returns: a list consisting of an integer and a vector space

Given a smooth, complete and projective toric variety with Coxring S and a f.p. graded S-module M, this method uses a theorem by Greg Smith to compute H^0 of the sheaf widetildeM on this toric variety. This is achieved by computing the degree zero layer of Hom ( B (e ), M ). The method computes this degree zero layer as a vector space and return a list consisting of the integer e and this vector space.

9.4-2 HiByGS
‣ HiByGS( variety, M, index )( operation )

Returns: a list consisting of an integer and a vector space

Given a smooth, complete and projective toric variety with Coxring S and a f.p. graded S-module M, this method uses a theorem by Greg Smith to compute H^index} of the sheaf widetildeM on this toric variety. This is achieved by computing the degree zero layer of Ext^index} ( B ( e ), M ). The method computes this degree zero layer as a vector space and returns a list consisting of the integer e and this vector space.

9.4-3 AllCohomologiesByGS
‣ AllCohomologiesByGS( variety, M )( operation )

Returns: a list consisting of an integer and of a list of vector spaces

Given a smooth, complete and projective toric variety X_Σ with Coxring S and a f.p. graded S-module M, this method computes all cohomology classes of widetildeM. To this end an integer e is computed such that the degree zero layer of Ext^index} ( B (e ), M ) is isomorphic to H^index} ( X_Σ, widetildeM ) for all i ∈ [ 0, dim ( X_Σ ) ]. The method returns this integer e and the collection of all degree zero parts (represented as vector spaces).

9.5 Cohomology of coherent sheaves: on CPn

9.5-1 H0OnProjectiveSpaceViaLinearRegularity
‣ H0OnProjectiveSpaceViaLinearRegularity( variety, M )( operation )

Returns: an integer

Given that variety is a complex projective space with Coxring S and M a f.p. graded S-module, this method computes the dimension of the vector space H^0 ( widetildeM ) and returns this integer. To achieve this an integer e is computed such that H^0 ( tildeM ) ≅ H_S^0 ( mathfrakm^e , tildeM )_0. This integer e in turn is computed by use of linear regularity of the module M.

9.5-2 H0OnProjectiveSpaceForAllTwistsViaLinearRegularity
‣ H0OnProjectiveSpaceForAllTwistsViaLinearRegularity( variety, M )( operation )

Returns: a function

Given that variety is a complex projective space with Coxring S and M a f.p. graded S-module, this method computes a function f : N_≥ 0 -> N_≥ 0 such that f ( n ) = dim ( H^0 ( widetildeM ( n ) ) ). This function is returned.

9.5-3 H0OnProjectiveSpaceInRangeViaLinearRegularity
‣ H0OnProjectiveSpaceInRangeViaLinearRegularity( variety, M, range )( operation )

Returns: a list of non-negative integers

Given that variety is a complex projective space with Coxring S and M a f.p. graded S-module, this method uses the function f : N_≥ 0 -> N_≥ 0 computed by \emph{H0OnCPNForAllTwistsViaLinReg} and evaluates it for all integers in \emph{range}. The resulting list of non-negative integers is returned.

9.6 Cohomology of coherent sheaves: H0

9.6-1 H0
‣ H0( variety, M )( operation )

Returns: a vector space or its dimension

Given that variety is a smooth and complete toric variety with Coxring S and M a f.p. graded S-module, this method computes H^0 ( widetildeM ). To this end it hands the input of one of the above presented methods. Which method the input is handed to depends on the input. In general however, the method that computes this cohomology class the fastest should be chosen.
